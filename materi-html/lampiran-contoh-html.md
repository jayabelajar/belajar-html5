## A. Contoh Lengkap Halaman HTML5
```html
<!DOCTYPE html>
<html lang="id">
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <meta name="description" content="Contoh halaman HTML5 lengkap dengan elemen semantik" />
  <title>Contoh Halaman HTML5 Lengkap</title>
</head>
<body>
  <header>
    <h1>Belajar HTML5</h1>
    <nav>
      <a href="#">Beranda</a> | <a href="#">Artikel</a> | <a href="#">Kontak</a>
    </nav>
  </header>

  <main>
    <article>
      <h2>Manfaat HTML5</h2>
      <p>HTML5 memberikan banyak fitur baru untuk membangun website modern tanpa plugin tambahan.</p>
      <figure>
        <img src="html5-logo.png" alt="Logo HTML5" width="120" />
        <figcaption>Logo resmi HTML5</figcaption>
      </figure>
    </article>

    <section>
      <h3>Contoh Formulir HTML5</h3>
      <form>
        <label>Nama:</label>
        <input type="text" placeholder="Masukkan nama" required />
        <label>Email:</label>
        <input type="email" placeholder="Masukkan email" required />
        <input type="submit" value="Kirim" />
      </form>
    </section>
  </main>

  <aside>
    <h3>Tips</h3>
    <p>Gunakan tag semantik dan validasi kode Anda di W3C Validator.</p>
  </aside>

  <footer>
    <p>&copy; 2025 Belajar HTML5 â€” Semua Hak Dilindungi</p>
  </footer>
</body>
</html>
```
